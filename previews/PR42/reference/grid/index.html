NODGrid

FerriteDistributed.NODGrid — Type
NODGrid{dim,C<:AbstractCell,T<:Real} <: AbstractNODGrid{dim}

Scalable non-overlapping distributed grid. This data structure is composed of a local_grid and full topological information on the process boundary, i.e. how vertices, edges and faces are connectedted between processes.

FerriteDistributed.NODGrid — Method
NODGrid(grid_comm::MPI.Comm, grid_to_distribute::Grid{dim,C,T})

Construct a non-overlapping distributed grid from a grid with the SFC induced by the element ordering on a specified MPI communicator. It is assumed that this function is called with exactly the same grid on each MPI process in the communicator.
